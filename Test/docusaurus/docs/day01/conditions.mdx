# Conditions & Loops (60 นาที)

- แนะนำ if/else, `===`, และแนวคิด truthy/falsy
- ลูป for เมื่อรู้จำนวนรอบ, while เมื่อใช้เงื่อนไข

```js
const score = 73;

if (score >= 80) console.log('A');
else if (score >= 70) console.log('B');
else console.log('ต้องฝึกต่อ');

for (let i = 1; i <= 3; i += 1) {
  console.log(`Loop ${i}`);
}
```
