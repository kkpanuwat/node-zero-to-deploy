# Conditions & Loops (60 นาที)

ใน 1 ชั่วโมงนี้เราจะฝึกใช้เครื่องมือที่เจอบ่อยสุด ๆ ในทุกภาษาโปรแกรม: เงื่อนไข (`if`, `else`, `switch`) และลูป (`for`, `while`, `forEach`). เพื่อให้เห็นภาพว่าพวกมันช่วยเราได้อย่างไร ลองมองผ่านมุมมอง “ร้านหนังสือ” ที่ต้องตัดสินใจและทำงานซ้ำ ๆ ทุกวัน

## เงื่อนไข: ตัดสินใจบนข้อมูลจริง

### if/else ขั้นพื้นฐาน

```js
const stock = 5;

if (stock > 0) {
  console.log('มีหนังสือพร้อมขาย');
} else {
  console.log('สินค้าหมด ช่วยแจ้งลูกค้าด้วย');
}
```

- `if (เงื่อนไข)` รันเฉพาะกรณีที่เงื่อนไขเป็นจริง
- ใช้ `else` หรือ `else if` เพื่อครอบคลุมกรณีอื่น ๆ

### เทียบค่าแบบเข้ม (`===`)

ใช้ `===` เพื่อเทียบค่าแบบตรงตัว ไม่แปลงประเภทให้เราอัตโนมัติ

```js
const memberType = 'PREMIUM';

if (memberType === 'PREMIUM') {
  console.log('รับส่วนลด 10% + ส่งฟรี');
} else if (memberType === 'STANDARD') {
  console.log('รับส่วนลด 5%');
} else {
  console.log('ลูกค้าทั่วไป ซื้อครบ 500 ส่งฟรี');
}
```

### Truthy / Falsy

ค่าใน JavaScript จะถูกแปลงเป็น Boolean อัตโนมัติ เช่น `0`, `''`, `null`, `undefined`, `NaN` นับเป็น falsy ส่วนค่าที่เหลือเป็น truthy

```js
const couponCode = '';

if (!couponCode) {
  console.log('ยังไม่ได้ใส่คูปอง');
}
```

## ลูป: ทำงานซ้ำโดยไม่เขียนโค้ดซ้ำ

### for loop – รู้จำนวนรอบล่วงหน้า

```js
const featuredBooks = ['Clean Code', 'Atomic Habits', 'Deep Work'];

for (let i = 0; i < featuredBooks.length; i += 1) {
  console.log(`จัดวางป้ายโปรโมต: ${featuredBooks[i]}`);
}
```

### while loop – ทำจนกว่าเงื่อนไขจะหยุด

```js
let remainingBoxes = 3;

while (remainingBoxes > 0) {
  console.log(`ย้ายกล่องหนังสือเหลืออีก ${remainingBoxes} ใบ`);
  remainingBoxes -= 1;
}
```

### forEach – วนสมาชิก Array แบบกระชับ

```js
const pendingOrders = [
  { id: 'A1', customer: 'โม' },
  { id: 'B2', customer: 'ผิง' },
];

pendingOrders.forEach((order, index) => {
  console.log(`#${index + 1} เตรียมจัดส่งให้ ${order.customer}`);
});
```

## โจทย์ฝึก

1. เขียนเงื่อนไขเช็กยอดซื้อ
   - ถ้า >= 1,000 ให้แสดง “ส่งฟรี”
   - ถ้าระหว่าง 500–999 ให้แสดง “ลด 5%”
   - น้อยกว่านั้นให้แสดง “รับคะแนนสะสมเพิ่ม”
2. สร้างโปรแกรมสุ่มรหัสหนังสือ 1–10 ต่อเนื่องด้วย `while` จนกว่าจะเจอเลข 7 แล้วบอกจำนวนครั้งที่สุ่ม
3. ใช้ `for` หรือ `forEach` เพื่อแสดงรายการหนังสือที่ต้องจัดออร์เดอร์ของแต่ละสมาชิก เช่น “สวัสดีคุณ A ขอแพ็กหนังสือ Book X”
4. (ท้าทาย) สร้างระบบลงชื่อพนักงานโดยใช้ `do...while` ที่บังคับให้พิมพ์ชื่ออย่างน้อยหนึ่งครั้ง และให้หยุดเมื่อพิมพ์ชื่อไม่ว่างเปล่า
