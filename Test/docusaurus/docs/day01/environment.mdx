# JavaScript Environment (30 นาที)

เป้าหมายคือให้ผู้เรียนเซ็ตเครื่องเพื่อรัน JavaScript บน Node ได้เองภายใน 30 นาที

## ขั้นตอน
1. ติดตั้ง VS Code + Node.js LTS (ตรวจด้วย `node --version`)
2. เปิด Terminal ในโฟลเดอร์คอร์ส แล้วสร้างไฟล์ `hello.js`
3. รัน `node hello.js` เพื่อยืนยันผลลัพธ์

```js
// hello.js
console.log('Hello JS!');
```

**Key Takeaway**: Node.js คือ runtime ที่ให้ JS ทำงานนอกเบราว์เซอร์ ลองแก้ข้อความในไฟล์แล้วรันใหม่เพื่อให้ชินกับ workflow
