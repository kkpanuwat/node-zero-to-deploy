# Functions + Arrow Functions (45 นาที)

- โครงสร้างฟังก์ชัน, parameter, return, default parameter
- เปรียบเทียบ function declaration และ arrow function

```js
function formatMovie(title, rating = 'N/A') {
  return `${title} (rating: ${rating})`;
}

const formatArrow = (title, rating) => `${title} - ${rating}`;

console.log(formatMovie('Interstellar', 8.7));
console.log(formatArrow('Inception', 8.8));
```
