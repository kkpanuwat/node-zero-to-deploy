# Module 1.1: Introduction & Environment Setup

เริ่มต้น เราจะเตรียมสภาพแวดล้อมสำหรับการเขียน Node.js ให้พร้อมใช้งาน เพื่อให้ทุกคนทำตาม Workshop ได้เหมือนกันหมด ขั้นตอนทั้งหมดจะใช้ NVM (Node Version Manager) ซึ่งเป็นตัวช่วยจัดการ Node.js แบบง่ายที่สุด ติดตั้งหลายเวอร์ชัน สลับเวอร์ชันได้ และอัปเดตได้สะดวกมาก

### เคส 1: โปรเจกต์เก่าใช้ Node 16 แต่คุณมี Node 20
clone โปรเจกต์จาก GitHub มาและกำลังจะ install package ต่าง ๆ:

```sh title="sh"
npm install
```

 สิ่งที่คุณจะได้พบคือ...
```sh title="sh"
Error: Cannot find module...
```

### เคส 2: มีหลายโปรเจกต์ที่ต้องทำงานในเครื่องเดียวกัน

| โปรเจกต์  | ประเภทงาน             | รายละเอียด                                      |
| --------- | --------------------- | ----------------------------------------------- |
| CoffeeHub | เว็บร้านกาแฟ          | ใช้ระบบเดิมที่ยังรองรับแค่ **Node 16**          |
| EduLearn  | ระบบเรียนออนไลน์      | ใช้ฟีเจอร์ใหม่ ES Modules → ต้องใช้ **Node 18** |
| ShopMania | เว็บขายของ E-commerce | Dev ทีมใหม่ใช้ **Node 20**                      |

- CoffeeHub รันไม่ได้เพราะ Node ปัจจุบันสูงเกินไป
- EduLearn ขึ้น error เพราะ dependency บางตัวไม่รองรับ Node 20
- ShopMania ต้องการ Node ใหม่กว่า แต่เครื่องคุณยังเป็น Node 16

### แล้วทำยังไงให้จัดการหลายเวอร์ชันได้ง่าย?

นี่คือเหตุผลที่เราต้องใช้ NVM (Node Version Manager) เพราะช่วยให้คุณสามารถ: ติดตั้ง Node หลายเวอร์ชันในเครื่องเดียว

สลับเวอร์ชันได้ทันที ตั้งค่า default สำหรับแต่ละ project ได้ ป้องกันปัญหา dependency พังเพราะเวอร์ชันไม่ตรง

### แล้วเราจะติดตั้ง NVM ยังไงล่ะ?

ตอนนี้เรารู้แล้วว่า ทำไมต้องใช้ NVM
คราวนี้มาดูวิธี ติดตั้งจริงบนเครื่องของเรา กันบ้าง
ซึ่งแต่ละระบบปฏิบัติการมีขั้นตอนต่างกันเล็กน้อย

#### การติดตั้ง NVM บน Windows

บน Windows จะใช้โปรแกรมที่ชื่อว่า nvm-windows
ไม่ใช่ตัวเดียวกับ macOS/Linus แต่ใช้งานเหมือนกันทุกอย่าง

⬇️ ดาวน์โหลดจาก GitHub (ไฟล์ .exe)

https://github.com/coreybutler/nvm-windows/releases

#### ติดตั้งตามขั้นตอนปกติ

- เลือก path ที่จะติดตั้ง NVM
- เลือก path Node เวอร์ชัน

#### เปิด CMD / PowerShell แล้วตรวจสอบว่าใช้งานได้

```sh
nvm version
```

#### ติดตั้ง Node เวอร์ชันที่ต้องการ

```sh
nvm install 18.20.2
nvm install 20.11.1
```

#### สลับเวอร์ชัน

```sh
nvm use 18.20.2
```

#### เช็คเวอร์ชัน Node ปัจจุบัน

```sh
node -v
```

#### วิธีติดตั้ง NVM บน macOS

#### รันคำสั่งติดตั้ง NVM
```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
```

#### โหลด environment เข้า Shell
```sh
source ~/.nvm/nvm.sh
```

#### ตรวจสอบว่า NVM ติดตั้งสำเร็จหรือไม่
```sh
nvm version
```
