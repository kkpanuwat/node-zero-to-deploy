# Module 1: JS Basics — Overview

ปูพื้น JavaScript สำหรับผู้เริ่มต้นสาย Backend ด้วย story “เมนูร้านอาหาร” ใช้ชุดตัวอย่างเดียวกันทุกบท เพื่อพร้อมต่อยอดเป็นเว็บเซิร์ฟเวอร์ในวันถัดไป

<img src={MultiProjectImg} alt="JS Basics" style={{ width: "100%", maxWidth: 960 }} />

## คุณจะได้เรียนอะไร (Day 1)
- Variables (let/const), การตั้งชื่อ, Mutability
- Data Types: primitive vs reference, Truthy/Falsy, typeof
- Functions: normal/arrow, default/rest params, this พื้นฐาน
- Objects & Arrays: copy vs reference, spread, destructuring
- Loop & Condition: for/while, break/continue, ternary
- Scope & Hoisting: var/let/const, TDZ, scope chain

## Chapters & Labs
1) Variables — คำนวณยอดขายร้านกาแฟ + VAT  
2) Data Types — สำรวจชนิดข้อมูลสินค้า/แท็ก, copy by reference  
3) Functions — calcVat, formatPrice, map/filter เมนู  
4) Objects & Arrays — cart รวมยอด, spread copy, destructuring  
5) Loop & Condition — ตรวจคะแนน/เงื่อนไข, reduce รวมค่า, ternary  
6) Scope & Hoisting — var/let/const ในบล็อก, hoist/TDZ, scope chain

## วิธีสอน (แนะนำลำดับ)
1) Warm-up: Input → Process → Output ด้วยตัวอย่าง “คิดราคาเมนู”  
2) Variables/Data Types: ใช้ราคาเมนู/สถานะร้าน (เปิด-ปิด) และ VAT  
3) Functions: เขียน `calcVat`, `formatPrice`, เปรียบ normal vs arrow  
4) Objects/Arrays: เก็บเมนู/ตะกร้า, โชว์ copy vs reference ด้วย spread  
5) Loop/Condition: รวมยอด, ตรวจส่วนลด, ternary สำหรับ label  
6) Scope/Hoisting: var/let/const ใน if/for, demo TDZ/ hoist

## Lab ต่อเนื่อง
- ฟังก์ชันจัดการเมนูใน Array: add/update/remove/search  
- ข้อมูลเมนูนี้จะถูกใช้ต่อใน Module 2 เพื่อทำ Web Server แรก

## เตรียมเครื่องมือ
- ติดตั้ง Node (แนะนำผ่าน NVM) และ VS Code  
- ถ้ายังไม่ตั้ง NVM หรืออยากสลับเวอร์ชัน ดูคู่มือใน Module 2 Chapter 2

## ผลลัพธ์เมื่อจบวัน
- อ่าน/เขียน JS พื้นฐานได้ เข้าใจ scope/hoisting  
- มี dataset เมนู + helper functions พร้อมส่งต่อให้ Node/Express ในวันถัดไป
