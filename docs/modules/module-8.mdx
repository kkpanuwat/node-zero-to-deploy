# Module 8: Upload & Search (Day 8)

เพิ่มความสมจริงด้วยการอัปโหลดรูปอาหารและค้นหาเมนู

## Chapter Highlights
- File upload ด้วย Multer (เก็บที่ `/public/uploads`)
- Search + Pagination (limit/offset)
- แสดงรายการเมนูพร้อมภาพ

## Lesson Flow
1) ติดตั้งและตั้งค่า Multer (disk storage, กำหนด dest และ filename)  
2) Route อัปโหลด: รับไฟล์ + body, เก็บ path ลง DB (Menu.imagePath)  
3) หน้า list เมนูแสดงภาพจาก `/public/uploads`  
4) เพิ่ม search (`?q=`) และ pagination (`page/pageSize` หรือ `limit/offset`)

## Lab
- อัปโหลดรูปเมนู + list เมนูพร้อมภาพ
- ค้นหา/กรองเมนูตามชื่อและแบ่งหน้า

## เป้าหมาย
- จัดการไฟล์อัปโหลดพร้อมเสิร์ฟเป็น static
- UX ดีขึ้นด้วยค้นหาและ pagination

## Implementation Notes
- ใช้ Multer รับไฟล์: เก็บชื่อไฟล์/พาธลง DB เพื่อใช้เสิร์ฟในหน้าเว็บ
- route search รับ query `?q=` และรองรับ `limit/offset` หรือ `page/pageSize`
- คุมสิทธิ์อัปโหลดผ่านหน้า admin (reuse session เดิม)
*- ตรวจขนาด/ชนิดไฟล์ก่อนรับ และลบไฟล์เดิมเมื่อแก้ไขเมนู (ถ้าต้องการ)
